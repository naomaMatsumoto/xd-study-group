

#### XD の勉強会

- リンクされたシンボル
  - ArtBoardと呼ばれている。
  - シンボルの登録を行い、何度も使用可能
  - リンクされているため、シンボルの編集は不可
    - 元のシンボルで編集は可能
    - 変更した際にリンクにカーソルを当てるといい感じに変わる。
    - バージョン変更でシンボルからコンポーネントになる。
      - 昨日拡大すると発表あり。
- 音声コントーロール
  - 音声をきっかけにアプリとして表示可能
- ドラッグジェスチャー
  - イベントにドラッグなどを使用してアニメーションを実現可能
  - イベントに時間などを指定して、自動アニメーションとすることが可能
  - 時間でドラッグやスクロールアニメーションなど実現可能となる
  - 文字は自分で変更可能
